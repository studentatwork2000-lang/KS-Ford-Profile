# K.S. Ford Static Site

Static multi-page site built directly from the Stitch export.

Routes:

- `/`
- `/books/`
- `/about/`
- `/contact/`

Editable files:

- `js/book-data.js` controls book preview content and confirmed buying links.
- `js/main.js` contains shared behavior for previews, mobile navigation, safe placeholder links, contact placeholders, and the About carousel.
- `css/site.css` contains additive shared styles for the modal and small interaction surfaces.
